Ubuntu 20.04LTS 文件夹图标丢失

之前在安装环境和依赖的时候，不知道用了什么操作，文件夹图标就消失了？？？

我寻思着我也没做啥，文件夹图标就丢失了，一直用命令行进行文件操作挺麻烦的。

运行命令：

sudo apt install gnome-shell-extension-desktop-icons

然后重新登入、登出，就能在应用列表看到了

最后右键Add to Favorites就行了
